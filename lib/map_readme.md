# 地图重要说明

### PolygonGeofencePage
**PolygonGeofencePage逻辑说明：**
1. 页面初始化时，会根据传入的`locList`参数，创建多边形围栏。
2. 当用户进入或离开围栏时，会触发`onRegionId`回调，返回触发的围栏ID。
3. 围栏的判断逻辑：
   - 当用户进入围栏时，会检查是否已经进入过该围栏，如果没有，则触发回调。
   - 当用户离开围栏时，会检查是否已经离开过该围栏，如果没有，则触发回调。
4. 围栏的判断是基于设备的位置和围栏的形状，所以不需要用户点击围栏即可触发回调。
5. 围栏的判断是基于设备的位置和围栏的形状，所以不需要用户点击围栏即可触发回调。
